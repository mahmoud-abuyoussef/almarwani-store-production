## 🚀 Project Overview

Almarwani Store is a Saudi-based eCommerce platform that previously had very limited visibility on search engines, with major indexing and technical SEO issues preventing it from appearing in Google search results.

This project focused on a complete SEO overhaul, including technical fixes, content optimization, and structural improvements to ensure full search engine visibility and better user experience.

The goal was to transform the website from a non-indexed platform into a fully discoverable and search-optimized eCommerce store.

---

## ✨ Key Improvements

- Fixed all Google Search Console indexing issues
- Resolved coverage errors and crawl problems
- Improved site architecture and URL structure
- Enhanced mobile performance and usability
- Optimized page speed and Core Web Vitals
- Implemented structured data (JSON-LD)
- Improved internal linking strategy
- Fixed sitemap and robots.txt configuration
- Enhanced metadata across all pages
- Improved Arabic keyword targeting strategy

---

## 🧠 SEO Strategy

The SEO strategy was built around three main pillars:

### 1. Technical SEO

- Fix crawl errors and indexing issues
- Optimize robots.txt and sitemap
- Improve site structure hierarchy
- Enhance page speed and performance

### 2. On-Page SEO

- Keyword optimization for Saudi market
- Optimized titles, meta descriptions, and headings
- Improved content structure and readability
- Implemented structured data markup

### 3. Off-Page & Authority Signals

- Improved search engine trust signals
- Strengthened internal linking structure
- Enhanced content discoverability
